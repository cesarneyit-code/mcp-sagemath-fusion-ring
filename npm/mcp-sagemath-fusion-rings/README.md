# mcp-sagemath-fusion-rings (npm wrapper)

This package is a thin `npx` wrapper around the PyPI package:
`mcp-sagemath-fusion-rings`.

## Usage
```bash
npx -y mcp-sagemath-fusion-rings
```

With explicit Sage binary:
```bash
SAGE_CMD=/path/to/sage npx -y mcp-sagemath-fusion-rings
```

## Requirement
You need `uv`/`uvx` installed:
https://docs.astral.sh/uv/
